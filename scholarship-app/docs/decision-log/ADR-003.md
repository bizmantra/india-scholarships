# ADR-003

Area: Architecture, SEO
Context: Including year suffixes in URLs breaks existing backlinks and resets search rankings every cycle.
Date: June 28, 2026
Decision Name : Evergreen URL Slugs (No Years)
Status: Accepted

### Context

Including year suffixes in URLs breaks existing backlinks and resets search rankings every cycle.

### Conversational Rationale

Slugs must remain static and evergreen. We established a strict rule: never put year suffixes (like -2025) in page URLs. When a rollover occurs, the URL stays exactly the same (e.g. /sitaram-jindal-foundation-scholarship), and we update the internal year metadata dynamically. This ensures our pages build domain authority over time, and any backlinks shared by students or schools remain valid forever.
